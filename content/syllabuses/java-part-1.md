## Java Part 1 & Part 2 Software Requirements
- Ensure the ability to install:
  - Java Development Kit (JDK)
  - Spring Boot
  - Git
  - Gradle
  - Maven
  - JUnit
  - PostgreSQL
  - Docker
  - Docker-compose
  - IntelliJ IDEA (preferred) / VSCode
  - JDBC Templates
  - JPA Hibernate
  - Log4J-2
  - Spring Boot (and any dependencies that Spring Boot requires) 
  - Thymeleaf
